# Quiz Flask Application

Welcome to the Quiz Flask Application! This is a simple web-based quiz application built using Flask, a lightweight WSGI web application framework in Python. The application allows users to take a quiz, answer questions, and receive their scores upon completion.

## Features

- A set of sample quiz questions with multiple-choice answers.
- Randomized question order for each quiz attempt.
- User score tracking and display at the end of the quiz.
- Responsive design with a clean and simple user interface.

## Requirements

To run this application, you need to have Python installed on your machine. You also need to install Flask. You can do this using pip:

pip install Flask

Getting Started
Clone the repository to your local machine:


git clone https://github.com/HamreethLS/Quiz-FLASK.git

Navigate to the project directory:

cd Quiz-FLASK

Run the application:

python app.py

Open your web browser and go to http://127.0.0.1:5000 to access the quiz application.
